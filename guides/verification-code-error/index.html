FAQ

Why does verification code error happen?

Expired code, wrong phone format, delayed SMS, or wrong flow can cause it.

Is this the same as OTP not received?

No. In this case a code exists but fails or expires.

What evidence matters?

Code screen, visible message, time, and phone format hint.
