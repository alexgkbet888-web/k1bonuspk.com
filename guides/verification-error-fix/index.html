Find the verification step

Verification errors can happen at phone number entry, OTP, device change, or account safety review. The fix depends on where the screen appears.

Network and cache context

A weak mobile connection or stale cache can make verification look broken. Compare mobile data and Wi-Fi before reinstalling.

Support evidence

The strongest support note includes time, device, network, phone format, and the exact verification message.

FAQ

Is verification error the same as login error?

Not always. Verification can fail before password or account login is checked.

Should I reinstall immediately?

No. Save the error message and check network/cache first.

What details help support?

Phone format, time, device, network type, and the exact verification screen.
